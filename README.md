<a href="https://bitstarz.netlify.app/#/" target="_blank">Demo Page</a>

## How to run
Clone this repository in a folder then:

``` bash
# Install dependencies
npm install

# run application in http://localhost:3000
npm run dev
```

## How to build

``` bash
# Files are available in build folder
npm run build
```
